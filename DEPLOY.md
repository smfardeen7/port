# Enable GitHub Pages (required for Deploy step)

For the **Deploy** job in Actions to succeed, turn on Pages and use **GitHub Actions** as the source:

1. Open **https://github.com/smfardeen7/port/settings/pages**
2. Under **Build and deployment** â†’ **Source**, select **GitHub Actions**
3. Click **Save**

After that, each run of "Build and Deploy to GitHub Pages" will build and deploy. Your site will be at **https://smfardeen7.github.io/port/**
